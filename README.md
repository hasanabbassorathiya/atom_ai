# Atom AI: The Neon Observatory

Atom AI is an on-device, private AI agent application built to transcend the standard chatbot experience. By leveraging the **RunAnywhere SDK**, Atom AI runs Large Language Models, Speech-to-Text (STT), Text-to-Speech (TTS), and Vision capabilities entirely offline, ensuring maximum data privacy and low-latency performance.

## Creative North Star: "The Neon Observatory"

The aesthetic is designed as a digital cockpit. It moves away from flat SaaS interfaces, opting for a layered, cinematic experience with deep charcoals and vibrant cyan/purple accents, emphasizing tonal depth and asymmetry to create an authoritative, hyper-modern interface.

## Key Features

- **Private On-Device AI:** Powered by the RunAnywhere SDK, all processing happens locally. No data leaves your device.
- **Agentic Voice Interaction:** Real-time speech-to-text and text-to-speech for seamless, hands-free conversation.
- **Vision Analysis:** Capture or attach images to receive detailed, local analysis of your surroundings.
- **Neon Observatory UI:** A bespoke design system featuring glassmorphism, ambient luminosity, and an editorial-grade layout.
- **Model Management:** Fully integrated model download and management system to customize your AI capabilities.

## Tech Stack

- **Framework:** Flutter with Riverpod for robust state management.
- **AI Engine:** RunAnywhere SDK (LlamaCpp, ONNX backends).
- **Design System:** Kinetic Intelligence Framework (Neon Observatory).
- **Architecture:** Clean Architecture with Service-oriented state management.

## Setup

1. Clone the repository.
2. Run `flutter pub get` to install dependencies.
3. Ensure Firebase CLI is configured for your environment.
4. Run the app: `flutter run`.
